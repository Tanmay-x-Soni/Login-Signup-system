# Login Signup System (PHP + MySQL)

A responsive Login and Signup system built using PHP and MySQL using MVC architecture.

## Features

✅ User Signup  
✅ MVC Structure  
✅ Password Hashing  
✅ Validation System  
✅ Error Handling  
✅ Responsive Design  
✅ Database Connection  
✅ Preserve User Input on Validation Errors  
✅ Field Specific Error Messages
